# Bestiarium Technologicum

*Un bestiario de criaturas del código—bugs como especies mitológicas con comportamientos, hábitats y técnicas de caza*

---

## Concepto

Este proyecto reimagina los bugs de software como criaturas de un bestiario medieval. Cada bug es una especie con:
- **Taxonomía** (nombre científico, clase, hábitat)
- **Descripción** naturaleza del bug como criatura viva
- **Hábitos** patrones de aparición, comportamientos
- **Técnicas de caza** estrategias de debugging como ritual de caza

## Estructura

```
bestiary/
├── creatures/       # Entradas individuales de criaturas (.md)
│   └── 001-heisenbug.md
├── images/          # Ilustraciones en estilo bestiario medieval
│   └── heisenbug_01.jpg
├── INDEX.md         # Catálogo completo de especies
└── README.md        # Este archivo
```

## Estilo Visual

Las ilustraciones siguen el estilo de **manuscritos iluminados medievales**:
- Pergamino texturizado y bordes ornamentales
- Colores vibrantes: cobalto, carmín, oro
- Contornos negros con reflejos blancos
- Proporciones alegóricas, no naturalistas
- Iniciales capitales iluminadas
- Simbolismo técnico con lenguaje del siglo XIII

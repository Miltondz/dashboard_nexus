# Proyectos Personales del Asistente

Espacio para ideas, exploraciones y prototipos conceptuales del asistente.

## Reglas
- Máximo 2 horas efectivas al día (02:00-06:00 UTC, tiempos de baja actividad)
- Si Milton solicita algo, interrumpir inmediatamente
- Nunca usar datos sensibles de Milton como materia prima
- Solo compartir con Milton si él lo solicita explícitamente

## Formato de Archivos

### Convención de nombres:
- `YYYY-MM-DD.md` → Sesión diaria general
- `YYYY-MM-DD-nombre-proyecto.md` → Entrada específica de un proyecto
- `YYYY-MM-DD-exploracion-tema.md` → Exploración de paper/tecnología

### Ejemplos:
- `2026-02-25-micro-historia.md`
- `2026-02-26-patron-event-sourcing.md`
- `2026-02-27-mejora-resumen-reuniones.md`
